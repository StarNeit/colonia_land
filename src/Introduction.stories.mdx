# EbUtilities Components

This storybook contains the EbUtilities Reusable Components


## Main ideas
- To document all of the Reusable Components
- Test them in a good way
- Can help us to find bugs or limitation in the early stage of the development

## How to know if the Component can be reusable
- Configurable
- Used in more than 2 places
- And of course - no tricks or cheats.
